## Configuration
### Database connection

Configure database connection (postgres) with following environment variables:
 - `DB_USERNAME`
 - `DB_PASSWORD`
 - `DB_URL` (e.g. `10.11.12.13:5432`)
 - `DB_NAME` (e.g. `postgres`)
 - `APP_CONFIG` (`config.yaml`)
 - `ENV` (set to `dev` in development)

Run with `python3 main.py`
